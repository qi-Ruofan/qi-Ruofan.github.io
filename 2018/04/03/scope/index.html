【javascript】作用域详解
发表于 2018-04-03 | 更新于 2020-09-06
JavaScript
作用域

作用域分类

  • 全局作用域
  • 局部作用域/函数作用域
  • 块级作用域（es6以后）
  • 动态作用域（this）

全局作用域

特性：

  • 在函数外声明的全局变量在任何地方都能拿到，两个不同的文件也能取到
  • 没用var声明的变量虽然也具备全局作用域但是严格意义上不是全局变量（不可被删除），而是全局属性（可被删除）
  • 在函数中没用var声明的变量，具备全局作用域。

局部作用域（函数作用域）

特性：

  • 函数外部无法主动访问函数内部的信息
  • 共享方式：return/闭包

块级作用域

特性：

  • 行程块级作用域后，外部无法访问到块内部
  • 必须和let或const配合，行程暂时性死区

动态作用域

特性：

  • 可以通过bind,apply,call改变作用域指向

扩展：

let & const

  • let和const声明的变量不允许重复定义
  • let和const声明的变量不具备变量提升
  • let和const具有块级作用域
  • let和const不能作为全局对象的属性来访问
  • const定义常量，定义后不能再被修改
  • const声明时必须赋值
